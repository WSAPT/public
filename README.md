# WSAPT 스터디
We Solve Algorithm Problem Together

알고리즘 문제를 함께 풀어보는 스터디입니다.

# 스터디 진행방식
풀어볼 알고리즘 문제를 정한뒤 1주일 전에 정하고,
문제를 풀어본 뒤에, 스카이프로 온라인 미팅을 진행합니다.

## 스터디 미팅 시작
1. 월요일 저녁 8시에, 스카이프로 스터디 미팅을 시작합니다. (참여 멤버의 상황에 따라서, 요일및 시간은 조절합니다.)
2. 서로의 문제풀이를 리뷰/토론

# 멤버 모집중입니다.
한 두분 정도 더 오셨으면 좋겠습니다.
